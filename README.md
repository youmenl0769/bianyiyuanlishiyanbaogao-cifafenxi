# 编译原理实验报告-词法分析

## 资源文件描述

### 一、实验目的
1. 理解词法分析在编译程序中的作用；
2. 掌握词法分析程序的实现方法和技术；
3. 用C语言对一个简单的子集编制一个一边扫描的编译程序，以加深对编译原理的理解，掌握编译程序的方法与技术。

### 二、实验内容
用C语言编写一个词法分析器。要求：
- 输入一个简单小程序，输出每个单词种别码。
- 能够识别以下内容：
  1. 关键字：`while`、`if`、`else`、`switch`、`case`
  2. 标识符
  3. 常数
  4. 运算符：`+`、`-`、`*`、`/`、`<=`、`<`、`=`、`==`、`;`
